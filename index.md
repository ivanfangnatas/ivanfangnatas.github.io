## üåü About RedStone

Welcome to the RedStone Oracles Monorepo! This repository contains the essential components for the RedStone oracles ecosystem. We aim to provide you with reliable and up-to-date data for your decentralized applications (dApps) and smart contracts. 

RedStone changes how we deliver on-chain data. Data is linked to a user's transaction and removed after use. This approach lowers gas fees and avoids relying on costly storage. 

To learn more about how RedStone works, visit our [documentation](https://docs.redstone.finance).

## üì¶ Available Packages

Here are the main components you can explore in this repository:

- **[oracle-node](packages/oracle-node/)**  
  This package serves as a reference implementation of the data provider node.
  
- **[cache-service](packages/cache-service/)**  
  This provides an implementation of the RedStone cache nodes.
  
- **[evm-connector](packages/evm-connector/)**  
  This package is for connecting with Ethereum virtual machines.

- **[protocol](packages/protocol/)**  
  This is the protocol for RedStone functionalities.

- **[sdk](packages/sdk/)**  
  This package provides a software development kit for integrating RedStone.

- **[eth-contracts](packages/eth-contracts/)**  
  Access Ethereum smart contracts relevant to RedStone.
